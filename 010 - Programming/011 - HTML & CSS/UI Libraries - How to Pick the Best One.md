# How To Pick the Best UI Library

## There are 3 main kinds


- Extensions of CSS (SASS, LESS, Tailwind)
- Behavior Libraries (HeadlessUI, Radix, React Aria)
- Style Systems (TailwindUI, DaisyIU, Mantine)
